# livs
## PHP-Livs: komponera måltider

Ett gränssnitt för Livsmedelsverkets databas över näringsämnen i syfte att underlätta komposition av måltider. Systemkrav:

* PHP 8
* SQLite3

## Installation

Ladda ned till din lokala webbservers dokumentrot.
